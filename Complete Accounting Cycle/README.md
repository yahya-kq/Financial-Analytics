# 📊 Accounting Cycle – Complete Financial Statements

Welcome to my **Accounting Cycle Project** repository!  
This project demonstrates a full accounting cycle — from **Journal Entries** to a complete set of **Financial Statements** — built using real transactions for a business called **XYZ**.

## 📌 What's Included

This project covers the entire bookkeeping and reporting process:

- 📒 Chart of Accounts
- 📝 General Journal Entries
- 📚 General Ledger
- ⚖️ Trial Balance
- 💹 Income Statement
- 🏦 Balance Sheet
- 💵 Cash Flow Statement

## 🛠 Skills & Concepts Applied

- **Microsoft Excel** – for building and organizing all statements
- **Accounting & Finance Knowledge** – including:
  - Double-entry bookkeeping
  - Journal entries and ledger posting
  - Trial balance preparation
  - Income Statement (Revenue, COGS, Operating Expenses, Net Profit)
  - Balance Sheet (Assets, Liabilities, Equity)
  - Cash Flow Statement (Operating, Investing, Financing Activities)
  - Working capital analysis
